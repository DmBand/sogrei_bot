# Чат-бот в Telegram для магазина строительных материалов


```Python 3.8```, ```библиотека "aiogram"```

### Идея проекта
Предоставить возможность пользователям выполнять мониторинг цен на строительные материалы в магазине "Согрей-М" г.Гродно
